# Configuration

## eTax Settings

Navigate to **eTax Settings** in Frappe to configure your API credentials.

### Required Fields

| Field | Description |
|-------|-------------|
| API Key | MTA eTax API key |
| TIN | Taxpayer Identification Number |
| Environment | Production or Sandbox |

### Multi-Entity Setup

eTax supports multi-entity configurations.

1. Go to **eTax Entity**
2. Create a new entity for each taxpayer
3. Configure API credentials per entity

## Testing Connection

Use the **Test Connection** button in eTax Settings to verify your credentials.
